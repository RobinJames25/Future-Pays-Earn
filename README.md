Project Name: Future Pay Earns

Overview A React-based fintech dashboard designed for tracking investments and managing recurring bills. The application features a modern, dark-mode UI with responsive layouts and secure authentication views.

Key Features

Secure Authentication: Custom login interface with validation and animation.

Scoped Styling: CSS is modularized to prevent conflicts between the Marketing Home, Dashboard, and Auth pages.

Responsive Design: Fully adaptive layouts for mobile and desktop users.

Tech Stack: React, Vanilla CSS (Scoped), Google Fonts.
